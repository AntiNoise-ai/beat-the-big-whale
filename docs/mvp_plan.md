# MVP Plan

Goal:
Build a public-data-first MVP that recommends London Tube stations and lines for advertising based on audience fit.

V1 inputs:
- industry
- budget tier
- optional emphasis: residents, workers, shoppers, tourists, students, families, commuters

V1 outputs:
- top stations
- top lines
- reasons for each recommendation
- score breakdown
- confidence notes

Core scoring dimensions:
- resident_density
- daytime_workers
- retail_intensity
- dining_intensity
- tourism_intensity
- office_intensity
- student_presence
- family_presence
- affluence
- interchange_score
- footfall_proxy
- zone_centrality

Near-term product position:
Transit audience intelligence tool, not exact media-buying ROI engine.
