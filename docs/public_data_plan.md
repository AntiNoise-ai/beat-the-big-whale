# Public Data Plan

Public-first sources to use:
- TfL station metadata
- ONS / NOMIS / London Datastore demographics and workplace data
- OpenStreetMap / Geofabrik place data
- public station usage proxies

Phase 1 target:
Create a reproducible feature pipeline that can replace the hand-curated sample station scores.

Phase 1 outputs:
- station reference table
- catchment definitions
- raw POI counts by category
- normalized station feature vectors

Not required yet:
- paid footfall data
- ad pricing feeds
- campaign conversion data
